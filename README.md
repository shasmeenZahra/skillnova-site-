# Skillnova

A multi-page website for **Skillnova** — one place for development, design, and tutoring services, built and maintained by [Shasmeen Zahra](https://github.com/shasmeenZahra).

🔗 **Live demo:** _add your live link here once deployed (Vercel / GitHub Pages)_

---

## ✨ What's inside

Skillnova brings together three service areas across a clean, animated multi-page site:

- **Development & AI** — websites, web apps, and custom AI chatbots/automation
- **Design & Video** — branding, social media design, and video editing
- **Tutoring** — one-on-one online lessons across all school subjects (grades 6–12), with a focus on Math

### Pages
| File | Description |
|---|---|
| `index.html` | Home — hero, quick overview of all services, testimonials |
| `about.html` | About — bio, teaching background, experience timeline, skills |
| `services.html` | Development & Design services, how-it-works, pricing, selected work |
| `tutoring.html` | Tutoring details — subjects, process, pricing |
| `contact.html` | Contact form, direct contact methods, FAQ |

### Shared assets
| File | Description |
|---|---|
| `style.css` | Shared design system used across every page |
| `script.js` | Shared functionality — starfield background, floating assistant, FAQ accordion, contact form handling |

### Features
- Animated starfield background
- Floating quick-reply assistant widget
- Floating WhatsApp button
- Contact form (opens a pre-filled email)
- FAQ accordion
- Social links (LinkedIn, GitHub, Instagram, WhatsApp, Email)
- Fully responsive, mobile nav menu

## 📁 Folder structure

```
skillnova-site/
├── index.html
├── about.html
├── services.html
├── tutoring.html
├── contact.html
├── style.css
└── script.js
```

All files sit at the same level — no subfolders — so the relative links between pages (`href="about.html"`, `href="style.css"`, etc.) keep working wherever it's hosted.

## 🛠️ Built with

- Plain **HTML, CSS, and JavaScript** — no frameworks, no build step
- Google Fonts (`Space Grotesk` + `IBM Plex Sans`)
- A canvas-based starfield animation

## 🚀 Running locally

No installation needed — it's a static multi-page site.

1. Clone this repository
   ```bash
   git clone https://github.com/shasmeenZahra/skillnova-website.git
   ```
2. Open `index.html` directly in your browser, **or**
3. Serve it locally for a cleaner experience:
   ```bash
   npx serve .
   ```

---

## 🌐 Deploying

Since it's a static site, it can be deployed anywhere for free:

- **Vercel** — import the repo, deploy instantly (all pages work automatically)
- **GitHub Pages** — go to *Settings → Pages*, set the branch to `main`, and save. Keep `index.html` as the homepage filename.
- **Netlify** — drag-and-drop deploy also works
